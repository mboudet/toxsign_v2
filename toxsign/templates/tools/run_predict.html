Signature prediction tool
This tool allows users to predict the properties of a chemical by feeding its signature to a neural network trained on the ChemPSy dataset. Predictions include whether the chemical is an endocrine disrupting, or its distances to several chemical clusters.
How to perform the prediction
Input data: Select the signature using the "Select signature" selector. Select the model you wish to use.
Output data: Model-dependent table.
